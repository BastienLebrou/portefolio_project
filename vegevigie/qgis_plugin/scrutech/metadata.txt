about=ScruTech wraps reproducible geodata pipelines as QGIS Processing tools. VegeVigie: search Sentinel-2, build a datacube, mask clouds, compute NDVI monthly composites, run per-pixel Mann-Kendall + Sen's slope trend and NDVI-anomaly drought stress, aggregate to zones. PAF (forest fire): compute the Wildland-Urban Interface — the frontier line and contact band between forest and built-up areas (débroussaillement / defence zone). The VegeVigie datacube tools need a Python environment with the datacube stack (see the plugin README) and outbound access to Microsoft Planetary Computer; the PAF interface tool only needs GeoPandas/Shapely.
