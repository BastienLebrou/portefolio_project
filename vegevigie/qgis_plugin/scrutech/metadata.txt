description=One-click vegetation trend & drought analysis over an extent, from Sentinel-2 (VegeVigie engine).
about=ScruTech wraps the VegeVigie geodata pipeline as turnkey QGIS Processing tools: search Sentinel-2, build a datacube, mask clouds, compute NDVI monthly composites, run per-pixel Mann-Kendall + Sen's slope trend and NDVI-anomaly drought stress, and aggregate to zones. Run everything from the one-click "Analyze extent" algorithm, or stage by stage ("Pipeline stages" group) — the stages share a run folder + manifest, skip work that is already done, and chain in the Model Designer. Result layers load pre-styled. Requires a Python environment with the datacube stack (see the plugin README); needs outbound access to Microsoft Planetary Computer.
