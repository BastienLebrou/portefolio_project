about=ScruTech is a hub that exposes the ScruTech projects as QGIS Processing tools. VegeVigie: search Sentinel-2, build a datacube, mask clouds, compute NDVI monthly composites, run per-pixel Mann-Kendall + Sen's slope trend and NDVI-anomaly drought stress, aggregate to zones. PAF (forest fire): the Wildland-Urban Interface — frontier line and contact band between forest and built-up areas (débroussaillement / defence zone). Écobuage: weighted multi-criteria controlled-burn aptitude (0-100 map + 3-class zoning) from aligned criterion rasters. The VegeVigie datacube tools need a Python environment with the datacube stack (see the plugin README) and outbound access to Microsoft Planetary Computer; the PAF tool needs GeoPandas/Shapely; the écobuage tool needs only GDAL/numpy (bundled with QGIS).
